# Selenium Java POM Framework

A scalable, modular test automation framework built with Selenium WebDriver, Java, and TestNG using the Page Object Model (POM) design pattern.


==========================================================================================================================

# Project Structure

src/main/java/base: Contains BaseTest.java for browser initialization and teardown.

src/main/java/pages: Contains page classes (e.g., LoginPage.java) representing application UI components.

src/main/java/utils: Utility classes for logging and configuration.

src/test/java/tests: Contains test scripts (e.g., LoginTest.java).

testng.xml: Test suite configuration file.

=======================================================================================================================


# Prerequisites


1.Java JDK
2. Apache Maven
3. Eclipse IDE

=======================================================================================================================

# Setup Instructions


1. Clone the repository.
2. Import as a 'Maven Project' in Eclipse.
3. Run mvn clean install to download dependencies.
4. Execute tests via mvn test or by running testng.xml.

=======================================================================================================================
